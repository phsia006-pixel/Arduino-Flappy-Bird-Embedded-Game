# Arduino Flappy Bird Embedded Game

## Overview
This project implements a Flappy Bird-style game on an Arduino Uno using a 1.8 inch ST7735 SPI TFT display, a 16x2 LCD, and a joystick module. The game includes animated graphics, joystick-based control, score tracking, collision detection, and game-over/restart logic.

## Hardware
- Arduino Uno R3
- ST7735 1.8 inch TFT LCD
- 16x2 character LCD
- Joystick module
- Breadboard and jumper wires
- 220 ohm resistors

## Software / Embedded Concepts
- AVR-based embedded programming
- Cooperative task scheduler
- SPI communication
- ADC joystick input
- LCD display control
- TFT sprite rendering
- Collision detection
- Game state machine logic

## Features
- Title screen, score display, and game-over screen on 16x2 LCD
- Animated bird and moving pipes on TFT LCD
- Custom bird sprite rendered at 2x scale
- Joystick input for flap control
- Gravity and collision-based gameplay
- Restart logic after game over
